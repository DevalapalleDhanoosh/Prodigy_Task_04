Grocery Store Simulation

This Java-based Grocery Store Simulation models a real-world grocery shopping experience with an automated customer and inventory management system. The project simulates customers with random shopping lists and budgets, allowing them to purchase items based on availability and budget constraints.

Key Features:

  • Automated Customer Generation – Customers are created dynamically with random shopping lists and budgets.
  • Inventory Management – Tracks stock levels, updates inventory, and prevents out-of-stock purchases.
  • Queue System for Checkout – Simulates multiple customers processing through a queue-based checkout.
  • Object-Oriented Design – Implements OOP principles for better modularity and maintainability.
  • Console-Based Interface – The simulation runs in the terminal, displaying real-time updates on store operations.

Exploratory Data Analysis of Brazilian E-Commerce Public Data

This project focuses on Exploratory Data Analysis (EDA) of the Brazilian E-Commerce Public Dataset by Olist to uncover customer purchasing patterns, sales trends, and business insights. The analysis is performed using Python, Pandas, Matplotlib, and Seaborn, with a structured approach to data cleaning, manipulation, and visualization.

Key Features:

  • Data Cleaning & Preprocessing – Handles missing values, duplicates, and data inconsistencies.
  • Statistical Analysis – Examines numerical and categorical features to identify key trends.
  • Customer Behavior Analysis – Analyzes purchase frequency, payment methods, and order trends.
  • Sales & Product Insights – Identifies top-selling products, revenue distribution, and seller performance.
  • Data Visualization – Uses histograms, scatter plots, heatmaps, and time-series graphs to represent insights.
  • Multi-Dataset Merging – Integrates multiple datasets for a comprehensive analysis.
